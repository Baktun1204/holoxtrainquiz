# HOLO RAIL — ホロライブ×日本の電車クイズ

メンバー×電車のアートワーク65枚を見て、車両形式を当てる4択クイズです。正解を選ぶと、車両の導入時期・運行の目的・路線・設備と、メンバーごとのマッチングの根拠が表示されます。

## GitHub Pagesへのデプロイ

1. このアーカイブを展開し、中のファイル一式を新しいGitHubリポジトリの最上位フォルダにアップロードします。
2. デフォルトブランチ名は `main` のままにします。
3. リポジトリの **Settings → Pages → Build and deployment → Source** で **GitHub Actions** を選択します。
4. **Actions** タブの `Deploy to GitHub Pages` ワークフローが完了すると、PagesのURLが表示されます。

以後、`main` ブランチに変更をプッシュするたびに自動で再デプロイされます。リポジトリ名にかかわらず画像が正しく表示されるよう、相対パスの設定を適用済みです。

## ローカルでの確認

Node.js 22以上が必要です。

```bash
npm install
npm run dev
```

デプロイ用の成果物を直接つくるには、次のコマンドを使います。

```bash
npm run build
```

生成物は `dist` フォルダに出力されます。

## 主なファイル

- `src/App.tsx`: クイズ画面とスコア計算
- `src/explanations.ts`: 65車両・マッチングの詳細解説
- `src/quiz-data.json`: メンバーと正解車両のデータ
- `public/artworks`: クイズ用アートワーク65枚
- `.github/workflows/deploy-pages.yml`: GitHub Pages自動デプロイ設定
